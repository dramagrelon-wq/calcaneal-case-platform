# Media attributions

The welcome page uses public educational media from Wikimedia Commons for prototype demonstration. Replace these with de-identified, authorized teaching materials before production use.

Do not upload identifiable patient images, hospital numbers, names, dates, QR codes, or unapproved clinical media to the public repository.
