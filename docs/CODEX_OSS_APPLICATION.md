# Codex For Open Source Application Notes

This project may be a candidate for OpenAI's Codex for Open Source program once it has public repository activity, real maintainers, and demonstrable open-source value.

## Program Fit

Calcaneal Case is intended to be:

- an open-source medical education and case-management tool for calcaneal fracture
- a structured imaging, measurement, classification, follow-up, and discussion platform
- a privacy-conscious tool for physician-to-physician teaching
- a foundation for multi-center research workflows
- a long-term project maintained by a clinical and technical team

## Why Codex Support Helps

## Application Narrative

Calcaneal Case helps orthopedic trauma doctors collect de-identified calcaneal fracture cases, improve screen-photo imaging quality, perform semi-automatic angle measurement, record fracture classification, manage follow-up, and discuss cases for teaching. The project begins with calcaneal fracture and the three-step reduction method, then grows through sustained clinical review, privacy-aware case workflows, and teaching-quality improvements.

The project needs maintainer automation because long-term value depends on continuous case-schema updates, privacy review, imaging-tool improvements, documentation, testing, and community issue triage. Codex would help keep the project moving even when clinical maintainers are busy.
